女神異聞錄3

女神異聞錄3/ペルソナ3/Shin Megami Tensei: Persona 3
奉伊莉莎白(P4U&PQ)籲(無誤)：ATLUS已經正式發出有關劇情洩露(劇透)的警告 - 任何故事模式和PQ第四迷宮後的動畫一律會被刪除，而且會被全身封印的被丟到兄貴FOE大群中。在此請各編輯者小心處理任何故事相關的捏他，如需劇透亦請善用反白，敬請注意。
以下條目含有部分的劇情，沒過接觸原作又不想在影時間內身陷塔耳塔洛斯的話，請不要繼續看下去。

目錄

解說編輯

  • ATLUS於2006年7月13日發售的PS2遊戲，女神異聞錄系列的第三集。主題顏色為「青色」。
  • 本作是『女神異聞錄』系列中開始出現較大變革的一代，出現了許多創新的系統和設定，以及一些原有的系統復活。
  • 劇情風格為後來所稱的「新日常系」還有不少疑似向EVA致敬的地方
    • 前半部分還算日常，後半部分氣氛急轉直下，沉重灰暗。
  • 本作為用手槍形態的召喚器向自己射擊來召喚Persona。
  • 2007年4月19日發售了有追加要素的『女神異聞錄3 FES』，平台為PS2。
    • 宣傳語為「私を手放さなければ、きっとまた会える」（「只要沒有放下我，我們一定再能遇見」）。而Persona3的官方網站中出現「如果沒有賣掉Persona 3的光碟，一定會有好事發生。」意義的預告。最後於2006年12月8日正式發表。
    • 「FES」是節慶（festival）的簡稱。收錄描述女神異聞錄3結束後的新故事「Episode Aegis（美版：The Answer）」，而正編「Episode Yourself（美版：The Journey）」追加各種各樣的新要素，亦可以繼承女神異聞錄3的一部份遊戲記錄就是猛將傳。
    • 本篇新增Aegis（アイギス/埃癸斯/艾吉絲/神盾）的專屬community「永劫」。
    • 但後日談大部份只是追加劇情，對本編沒有影響，既不能使用本編裡的Persona全書，Commu也沒有繼承，而且是使用Aegis進行操作，通常攻擊手段限是貫通系。
    • 收錄了特別課外活動部的過去回想事件。
  • 2009年11月1日發售移植版『女神異聞錄3攜帶版』，平台為PSP。
    • 系統更加便利。新增技能卡貼紙條系統，每個PERSONA自帶一個，還可以去神社花5天複製或者去商店合成。
    • 新增了可以原地復活30次的最易等級「BEGINNER」和不能二週目繼承的最難等級「MANIACS」。
    • 也有被認為是劣化的地方：白天活動的地圖角色小動畫全消除，和一般AVG一樣靠移動光標才能選中目標還不如本身類型就算是AVG的某作
    • FES版啟用的各種過場動畫基本消除。直接導致不燃
    • 新增女性主人公路線。雖然主線劇情類似，但COMMU的內容一半以上都有了變化。
      • 即便和男主角路線COMMU一致的角色其實變化也很大。許多角色個性得到了進一步的補完。
      • 但也有個別極端的老粉表示FES才是完全版，而拒絕接受PSP版，認為P3P的性格補完其實是角色出格的表現，這就是仁者見仁了……
    • 2022年，Atlus宣布本作將與P5R一同移植到XBox、PC與NS

故事編輯

故事時間為2009年4月。10年前因事故而失去雙親而由親戚養育的主角，因為家庭的事情而轉校到小中一條龍學校「私立月光館學園」的高中部中。
在入住學生宿舍後不久，被奇形怪狀的怪物「陰影」襲擊，令隱藏在他（她）體內Persona的能力甦醒。因為這個契機，主角被迫知道這個被掩蓋的世界真實的一端。
那就是「影時間」，一天與另一天之間存在著的隱藏時間，「陰影」就是這個影時間的居民。在影時間中，普通人會變成棺柩形的形態，失去這時的記憶（這稱為「象徵化」）。所以大部分的人都不知道影時間的存在。
但有一少部份人，在影時間還保持著人的形態，那些人會被影吃掉他們的精神，而成為無力狀態（影人）。
在月光館學園理事長幾月修司下，組成使用Persona的影討伐部隊「特別課外活動部」。得到特殊Persona能力的主角被部長桐條美鶴的看中，成為隊員與特別課外活動部的隊長與危害人類的影作戰。

    • 關聯COMMU為「法王」的角色。
    • 巌戸台駅商店街上經營古書店「書蟲」，年過八十的老夫婦。姓氏是北村。
    • 會經常給主角各種零食。
    • 失獨夫婦，身為月光館教師的兒子因交通事故去世。
    • 爺爺在PSP版是塔耳塔洛斯失蹤人員之一。如果不在時限前救他，即便COMMU MAX也沒有後日談
  • 宮本一志 ＣＶ：私市淳
    • 主角為男性的情況下，關聯COMMU為「戰車」的角色。
    • 本來是運動社團的得力幹將，但因訓練過度而受傷。為了不讓周圍人知道，還在偷偷訓練。
    • 和西脇結子是青梅竹馬的關係，被對方照顧的無微不至。
    • 對由加莉有好感。
    • 女主角路線中和西脇結子常常一起出現。
  • 岩崎理緒 ＣＶ：早見沙織
    • PSP版新增角色。
    • 主角為女性的情況下，關聯COMMU為「戰車」的角色。
    • 月光館學園高等部2年級，主角加入的運動社團2年級領隊。傲嬌。
    • 嚴於律己也嚴待他人，和隊員產生了隔閡。
    • 暗戀青梅竹馬的友近。
  • 伏見千尋 ＣＶ：前田愛
    • 主角為男性的情況下，關聯COMMU為「正義」的角色。
    • 性格內向，憧憬少女漫畫一樣的展開。但是有男性恐怖症。
    • 美人。在玩家中人氣也不低。
    • 可以和男主角結為戀人。
    • 在女神異聞錄4中，擔當了學生會長。
  • Y子
    • 主角為男性的情況下，關聯COMMU為「隠者」的角色。
    • 主角打順平給的網遊時認識的網友。
    • 因為是網友，所以只有節假日才能交流。
    • 網名來源是初代女神轉生的弓子。
    • 聊到最後，主角會得知對方的真實身份是班導鳥海老師
  • 長谷川沙織
    • PSP新增角色。
    • 主角為女性的情況下，關聯COMMU為「隠者」的角色。
    • 主角所屬的文化社團的成員。和主角同年級實際上年歲比主角大
    • 穩重溫和的大姊姊，但性格太過老好人，和周圍人實質上無法正常交流。
  • 平賀慶介 ＣＶ：立花慎之介
    • 主角為男性的情況下，關聯COMMU為「命運」的角色。
    • 月光館學園3年級，主角所屬文化社團的部長。有點弱氣。
    • 是大醫院家的兒子，因為前途已被父母決定，對此感到不滿。
    • 女主角的路線裡是路人，加入的是攝影部。
  • 西脇結子 ＣＶ：佐藤聰美
    • 主角為男性的情況下，關聯COMMU為「剛毅」的角色。
    • 主角所屬的運動社團的經理人。
    • 喜歡照顧人，喜歡小孩。宮本一志的青梅竹馬，把他照顧的無微不至，時常跟他一起行動。
    • 初中時因為被籃球部的帥哥告白而招致兩名女生的討厭小木曽雪菜
    • 可以和男主角結為戀人。
    • 女主角的路線裡也大活躍，在理緒相關劇情裡是重要配角。
  • 大橋舞子 ＣＶ：折笠富美子
    • 關聯COMMU為「倒吊男（刑死者）」的角色。
    • 在長鳴神社公園裡出現的小學女生。
    • 因父母鬧離婚而陷入不安。
    • PSP版是塔耳塔洛斯失蹤人員之一，如果不在時限前救她，即便COMMU MAX也沒有後日談
  • 貝貝（ベベ） ＣＶ：菅沼久義
    • 關聯COMMU為「節制」的角色。
    • 全名為安德烈･羅蘭･讓･傑拉爾（アンドレ･ローラン･ジャン･ジェラール）。
    • 來自法國的留學生，時尚同好會會長。因受到叔母的資助隻身來日。
    • 愛稱Bebe是法語的Baby之意。
    • 喜愛日本文化。遣詞造句相當古風。用的都是片假名。
    • 女主角和他一起行動時，會製作出各種手工物件，可用於送禮。
  • 田中社長（たなか社長） ＣＶ：島田敏
    • 關聯COMMU為「惡魔」的角色。
    • 電視購物商城「時價網田中」的社長。
    • 說話方式有些接近女性。
    • 角色外形和名字的原型是擔當劇本的田中。
    • 除COMMU外，週日在家點擊電視，他會出來做廣告，主角可以購買當期的產品。
  • 無達 ＣＶ：佐藤正治
    • 關聯COMMU為「塔」的角色。
    • 破戒僧，明明是和尚卻混跡于酒吧，抽煙喝酒。
    • 喜歡說教，思想有些悲觀。
    • 本作第一傲嬌（無誤）。
  • 早瀬護
    • 主角為男性的情況下，關聯COMMU為「星」的角色。
    • 主角所屬的運動項目中，其他學校的超高校級王牌運動員。
    • 目前面臨家庭貧困的現實，正在拼命工作維持家計。
  • 末光望美
    • 主角為男性的情況下，關聯COMMU為「月」的角色。
    • 名字女性化，其實是男性，胖子。公認的美食之王。
    • 顏藝帝。
    • 劇場版中以背景角色的身份露了個臉，由於太好認反而容易引起注意。
  • 神木秋成 ＣＶ：神谷浩史
    • 關聯COMMU為「太陽」的角色。
    • 時常坐在長鳴神社裡的絕症病弱青年，時日無多。得的是家族遺傳病。
    • 要幫他找到筆才能跟他交流。
    • 由於姓為神木，不少玩家以為他就是神社裡那棵大樹
    • 劇情提示：走完他COMMU的玩家務必在畢業典禮前夕完成他的最後劇情，本路線最殺的地方在這裡。

一般人協助者編輯

  • 黑澤警官（くろさわ）
    • 在ポロニアンモール（桐葉商店街）地區擔當巡查的警官。撲克臉。
    • 只是普通人，沒有影時間的適應能力。
    • 和真田是舊識，能賣給主角各種裝備。週一會打點折
      • PS2版（無印和FES）晚上因為出去執行巡查公務所以不在，PSP版則是晚上照常營業。
    • PSP版還負責把救回來失蹤人口的報酬轉交給主角。
    • 不清楚影時間和SHADOW的真相，但是和活動部很有默契，對主角一行的行為充滿信任。
    • PSP女主角路線中，荒垣學長不死的關鍵人物
  • 青鬍子藥房店主（青ひげファーマシーの店主）
    • 在ポロニアンモール（桐葉商店街）地區開藥房的店主。
    • 週六會有大酬賓，便宜點。
    • 是個爽朗的大叔。
    • 除了藥物之外也會賣貓糧 完成天鵝絨房某個任務的必要道具
  • 古董店眞宵堂店主（古美術眞宵堂の店主）
    • 6月份開業的古董店眞宵堂店主，擁有銳利目光的身著白衣戴眼鏡的成熟女性。
    • 業務隨著遊戲的進化越拓越寬
      • 無印中可以用迷宮裡打到的寶石跟她交換提升PERSONA能力點數的卡片、高級消耗道具或禮物。
      • FES開始可以用武器素材[17]加強力的PERSONA去拜託她製作武器。
      • PSP版可以用寶石交換各種攻擊道具。
    • 其實是前桐條手下的暗影研究員

天鵝絨房編輯

    • 天鵝絨房的主人，在遊戲裡幫助玩家進行Persona的合體。
    • 從PERSONA一代開始就貫穿於整個系列作。

    • 天鵝絨房的住人，伊格爾的助手之一，三姐弟的次女。
    • 在PS2版和男主角路線裡是固定助手，在女主角路線中可選擇。
    • 相當脫線，嚴重缺乏常識。
    • 欺負弟弟為樂
    • 沒有COMMU，但仍然可帶回家（無誤）
      • 玩家走的是女主角的話，可以看到官方的終極病氣。
    • 迷宮隱藏BOSS之一。
      • 4代裡表明她為了救男主角而離開天鵝絨房了
  • 迪奧多（テオドア） ＣＶ：諏訪部順一
    • PSP新增角色，在女主角路線中可選擇。
    • 天鵝絨房的住人，伊格爾的助手之一，三姐弟的三男。
    • 優雅的紳士，但另一方面相當脫線，嚴重缺乏常識。本作第一耍笨役。
    • 會被兩個姐姐疼愛（讀作欺負）
    • 沒有COMMU，但仍然可帶回家（無誤）
    • 　迷宮隱藏BOSS之一。
    • 在之後的PERSONA Q裡也有登場待遇比哈姆子好多了
  • 瑪格麗特（マーガレット） ＣＶ：大原沙耶香
    • 天鵝絨房的住人，4代是伊格爾的助手，三姐弟的長女。
    • 在3代PSP版又出現，擔當隱藏之門的嚮導以及本遊戲最強隱藏BOSS

BOSS編輯

  • 榊貴隆也（タカヤ） ＣＶ：神奈延年／演出：西山丈也
    • Persona：許普諾斯（ヒュプノス）
      • 許普諾斯：希臘神話的睡神。
    • 武器：史密斯威森M500左輪手槍 / 攻擊屬性：貫通 / 秘儀：命運
    • 史特雷加最自信和魅力的年輕領袖。他作為主角的襯托，領導對抗Persona使用者小組和認為自己是一個虛偽的救世主。
    • 除了Aegis，他是唯一能夠在本代遊戲中不使用手槍召喚Persona的使用者：以手藉由抓牢自己的頭而且落到膝部從而召喚它。
      • Persona使用者在歷代遊戲中也沒有使用Evokers，但是召喚他們的Persona從不引起他們任何痛苦。
    • 雖然主張不回首過去、活在當下，卻也認定只有死亡才是真正的救贖。
    • 遊戲後期創建了一個邪教組織。
    • 角色外型讓人一眼就想到某個人。
  • 白戶陣（ジン） ＣＶ：小野坂昌也／演出：松本祐一
    • Persona：摩羅斯（モロス）
      • 摩羅斯：希臘神話的命運和死亡之神。
    • 武器：手榴彈 / 攻擊屬性：無 / 秘儀：隱者
    • 史特雷加的成員，參謀定位。關西腔。
    • 隆也的得力助手兼戰術顧問，是一名熟練的電腦駭客，在網路上擁有極高知名度與人氣。
    • 相當崇拜隆也與強烈支持他的目標。他是忠於隆也為「顯示出他的黑暗」，並幫助他通過桐條集團的實驗。
      • 不過他是理性的，必要時足以限制隆也的行動。每當隆也做出魯莽的行為，仁在那裡阻止他並且使他想起他們在那裡真正的目的。
    • 另一方面，他經常無意中幫助特別課外活動部以透露重要信息。
    • 官方小說「暗影悲歌」的主角，其作者稱仁是這三人中最有生活常識的人。
  • 吉野千鳥（チドリ） ＣＶ：澤城美雪／演出：はねゆり
    • Persona：美狄亞（メーディア）
      • 美狄亞：阿爾戈英雄遇到的女巫，因愛上伊阿宋而幫助他找到金羊毛。
    • 武器：鏈式手斧 / 攻擊屬性：斬擊 / 秘儀：刑死者
    • 史特雷加的成員，響導定位。
    • 白色哥德蘿莉服，有自殘傾向和藝術天分。
    • 持有的Persona美狄亞擁有將生命給予他人的能力。
    • 與順平是戀人未滿的關係，讓順平心靈成長的關鍵。
      • 自P3 Fes開始只要幾個關於順平的選向正確的話，就能改變她的命運。
  • 收割者（刈り取る者）
    • 通称「死神」。不是劇情BOSS，而是迷宫隱藏BOSS。
    • 出现條件如下：
      • 通常情況下，在一層迷宮中呆太久。就會出現（迷宮嚮導會有提示）
      • 戰鬥結束後的抽卡時間抽到了帶有死神的卡，會提前出現。
      • 沒有敵人的迷宮層會提前出現。
      • 全都是金色敵人的迷宮層會提前出線。
    • 本遊戲中唯一對應「死神」大秘儀的暗影。樣子為渾身黑衣，左眼部分覆蓋著壞掉的假面，雙持長火槍。初期非常難打，後期就容易多了。
    • 活動部發家致富的重要的經濟來源[18]
    • 4代和5代也有登場。
  • 尼克斯（ニュクス） ＣＶ：石田彰
    • 最終BOSS。
    • 名稱來自希臘的夜之女神。
      • 為什麼一個女神會發出男性的聲音?!
    • 也就是本作中的死之宣告者，被稱為集體意識的集合體，會導致地球上所有生物的死亡。
    • 戰鬥中會隨著階段改變自身的秘儀，依序照著編號從愚者到死神。
      • 每個秘儀會有不同的屬性和攻擊模式。
    • 戰鬥時的BGM中人聲部分的旋律和天鵝絨房間BGM的人聲旋律相同。
      • 5代中此BGM作為穿上DLC的天鵝絨房間服的一般戰鬥時使用所以說為什麼一個最終決戰的BGM會用在雜兵戰上。
  • 厄瑞玻斯（エレボス）
    • FES的最終BOSS。
    • 名稱來自希臘神話中黑暗的化身，為樓上那位的哥哥。
    • 人類的悲傷和負面情緒的表現，不斷嘗試解除尼克斯的封印。
    • 注意到S.E.E.S.的存在後將目標轉為艾吉絲，希望透過她解除封印。
    • 在P4U也有出場。
